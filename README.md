# 🚀 Selam! Ben Aras 👋

Gelişmiş web teknolojilerine tutkuyla bağlı bir Full-Stack Geliştiriciyim. Temiz, ölçeklenebilir ve güvenli kod yazmayı felsefe ediniyor; fikirden sunucuya (deployment) kadar projenin her aşamasında mimariyi kuruyorum. 

Şu sıralar R10.net topluluğu için faydalı scriptler kodlarken, bir yandan da yerel yapay zeka modelleri için web tabanlı kullanıcı arayüzleri (UI) ve mobil cihazlardan uzaktan erişim sistemleri geliştirmek üzerine heyecan verici projeler üzerinde çalışıyorum.
